# Open Docs

Docs for OpenClaw. Ask questions in plain English and get answers grounded in the real OpenClaw documentation, instead of hunting through pages.

**Live:** https://aidencullo.github.io/openclaw-rag-chatbot/

## Why

OpenClaw's setup has a lot of moving parts. Open Docs is a little chatbot that sits on top of the docs so you can just ask.

## Where this might go

Eventually: paste any project's docs URL and get a chatbot for that project out of it. Something like that probably already exists (Mintlify, kapa.ai, Inkeep, and a handful of others) — so for now this is more of an own-the-stack / learning exercise than a new idea.
